# Recreate GForm using React and React Hook Form

Aplikasi ini merupakan form survey perokok dan menampung hasil ke dalam Local Storage dan menggunakan Hook Form untuk validasi apakah input sesuai dengan yang diharapkan atau tidak.

## Cara Penggunaan Aplikasi
1. Clone Repo ini
2. Jalankan terminal
3. Tuliskan "npm run dev"
4. Click link localhost yang diberikan
5. Langsung gunakan aplikasi
